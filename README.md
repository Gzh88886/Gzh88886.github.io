# Gzh星轨塔罗 | 手机端交互型运势占卜网页
一款功能丰富、视觉生动的手机端塔罗牌占卜网页，支持多维度运势解读与触摸交互。


## ✨ 核心功能
- **4大占卜类型**：爱情/学业/事业/生活，覆盖日常场景
- **3牌阵运势算法**：当前状态+潜在影响+建议方向，生成专属指引
- **触摸交互体验**：牌面翻转动画、点击切换正反、历史记录本地存储
- **动态视觉设计**：星轨能量风格+加载动画，贴合年轻用户审美

## 🎨 特点
- 响应式布局：完美适配手机触摸操作
- 独立解读逻辑：每张塔罗牌对应不同类型的专属文案
- 轻量无依赖：纯HTML/CSS/JS开发，无需额外框架
